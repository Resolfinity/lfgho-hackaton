## How to run

Foundry, Node.js, Npm should be installed.

```
git clone git@github.com:Resolfinity/lfgho-hackaton.git

yarn

yarn forge-build

yarn typechain

```

Copy .env.example to .env and fill in all the vars.

To run tests, `anvil && yarn test`

To run relay, `yarn dev`
